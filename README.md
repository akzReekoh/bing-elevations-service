Bing Maps Elevations Service Plugin for the Reekoh IoT platform. Integrates a Reekoh instance to Bing Maps Elevations API to get elevations of places on Earth based on given coordinates.

## Description
This plugin gets the elevation(in meters) of a certain point based on input(latitude & longitude) using Bing Maps Elevations API.

## Configuration
To configure this plugin, a Microsoft account is needed to provide the following:

1. API Key - The Bing Maps API key to use.

These parameters are then injected to the plugin from the platform.
